mempool_init

该函数使用静态的方法初始化内存池，内存池以及内存池空间由使用者提供：

参数 说明
mp 内存池句柄
name 内存池名称
start 内存分配起始地址
size 内存池大小
block_size 内存块大小
num 内存块数量

mempool_alloc

该函数从内存池中申请一块内存

参数 说明
mp 内存池句柄
返回 说明
非NULL 分配的内存块地址
NULL 分配失败

mempool_free

释放一块由mempool_alloc()分配的内存块地址

参数 说明
block 内存块的地址
返回值 说明
无 无
