rvcc学习笔记11

支持return语句

​ return是一个关键字，用来返回具体值。在终结符的数据结构种类中，增加一个关键字终结符TK_KEYWORD，同时AST节点的种类也要增加一种ND_RETURN。

词法分析

​ 词法分析针对新增的关键字终结符，在完成终结符解析后，循环遍历将为关键字的终结符设为TK_KEYWORD。

语法分析

​ 文法改变，主要是stmt = “return” expr “;” | exprStmt。

代码生成

​ .L return: 设置return段标签。此后遇到return语句，用j offset无条件跳转指令跳转到return段。

测试

测试用例：

assert 3 ‘a=3;return a;’

assert 8 ‘a=3; z=5;return a+z;’
assert 6 ‘a=b=3;return a+b;’
assert 5 ‘a=3;b=4;a=1;return a+b;’

assert 3 ‘foo=3; foo;’
assert 74 ‘foo2=70; bar4=4; foo2+bar4;’
assert 3 ‘foo=3;return foo;’
assert 74 ‘foo2=70; bar4=4;return foo2+bar4;’

assert 1 ‘return 1; 2; 3;’
assert 2 ‘1; return 2; 3;’
assert 3 ‘1; 2; return 3;’
