c语言复习day3

1、指针的兼容性

​ 指针类型不一样，不能直接赋值。 例如ar1是指向由3个int值构成的数组，而pa也是指向3个int值构成的数组 ，因此pa和ar1的类型是一致的，和ar2的类型不一致。

​ 在函数中声明二维数据的方式：

​ void somefunction(int (*p)[4]); p是一个指向由4个int值构成的数组的指针

​ void somefunction(int p[][4]); 空的方括号表示p是一个指针

2、数组与指针的区别

​ 其主要区别在于heart是一个常量，而head是一个变量。两者都可以使用数组符号，也都可以使用指针加法，但是只有指针可已使用自增运算符。而且 head = heart 是正确的，但是heart = head是错误的。

3、常见的字符串函数

​ strlen：得到字符串的长度。

​ strcat：拼接两个字符串，strcat（str1，str2）

​ strncat: 在strcat的基础上限制了长度，因为strcat没有考虑第一个字符数组是否能够容纳下第二个数组。 strncat(str1,str2,21)

​ strcmp和strncmp: 比较连个字符串的内容，strncmp可以比较字符串的前n个字符。

​ strcpy和strncpy： 赋值字符串到一个数组中。 strcmp(str,temp) ,strncpy可以限定复制内容的长度，比较安全。

​ sprintf: 将几个字符串格式化的写入一个字符串中，

​ atoi和strtol: 将字符串转化为帧数。 value = strtol(str,end,10) // 以十进制转化str，end是转化后剩余的字符串

4、fseek()和ftell() 寻找文件中的位置

​ fseek() 由三个参数，fp文件指针，offset偏移量，还有一个起始点。

​ SEEK_SET 文件开始位置

​ SEEK_CUR 文件当前位置

​ SEEK_END 文件结尾位置

​ ftell() 返回一个long类型，他是用来寻找文件当前位置。

5、在结构中存储字符串，要用字符数组而不是指针。

​ 结构体不会给指针事先分配空间，他只是存储指针地址，指针没有初始化，这个地址可能是任何值。给结构体中的字符指针赋值，输入的内容可能会存储在内存的任何地方。

6、联合体（union）是一个能在统一个存储空间里（但不同时）存储不同类型的数据类型。

7、c语言中奇特的声明

8、位运算的用处

8.1、掩码

​ 设置一个mask与你想要进行运算的数进行与运算。最后得到结果。

​ mask = 0x3ff（11111111）

​ ch &= mask 无论最初的ch是8位还是16位，最终都会被修整成一个字节。

8.2、关闭位

​ mask = 2 （00000010）

​ flag &= ~mask (11111101)

​ 把flag中的位1变成0

8.3、转置位

​ mask = 2 （00000010）

​ flag ^= mask

8.4、查看一位

​ mask = 2 （00000010）

​ if((flag&mask)==mask){

​ puts(“wow!”);

​ }

8.5、右移运算符 >>

​ 在负数>>的情况： -2 右移两位。 在计算机中 数据都是用补码进行计算的。

​ -2 1000 0010 补码为 1111 1110 。

​ 右移两位变成 1111 1111 原码为1000 0001 所以 结果为 -1。

c语言复习day2

1、C语言文件操作

FILE *fp=fopen("a.txt","rw"); 打开文件,rw是对文件进行读写操作。

while((ch=(getc(fp)))!=EOF){

​ putchar(ch);

} 打开文件从文件中一个字符一个字符的读取并打印

2、函数递归

​ 一个函数自身调用自身的过程叫做递归。层层执行，逐级返回。函数中必须包括能终止递归调用的语句。

2.1 尾递归

​ 把递归调用语句放到函数结尾即恰好在return语句之前。

2.2 递归的四项基本法则

​ 1、基本情形，不用递归也能解出

​ 2、不断推进，对于那些需要递归求解的情形，每一次递归调用都必须要使求解状况朝接近基准情形的方向推进。

​ 3、设计法则，假设所有的递归调用都能运行。

​ 4、合成效益法则，在求解一个问题的同一实例时，切勿在不同的递归调用中做重复性的工作。

3、指针

​ 一个其数值为地址的变量。

​ ptr = &pooh /*把pooh的地址赋给ptr*/

​ val = * ptr /*得到ptr的值*/

​ int sh[20] 定义一个数组，sh=&sh[0], 数组名是数组首元素的地址，sh+index是元素sh[index]的地址，*(sh+index)是这个元素的值，等价于sh[index]。

​ 在设计一个函数时，传递的参数是数组，则其函数声明如下所示:

​ int sum(int * mar); mar是一个指针变量，发生调用是被赋值为数组的首地址

​ 调用的形式 total = sum(marbles)

以下的形式是等价的：

int sum(int *ar,int n);

int sum(int *,int );

int sum(int ar[],int n);

int sum(int [],int );

指针的常见操作：赋值，求值或取值，去指针地址，将一个整数加给指针，增加指针的值，从一个指针中减去一个整数，减小指针的值，求差值。

4、最大子序列和问题

​ 采用分治思想，使用递归来实现。将主问题划分为三个子问题，最大子序列和可能出现在前半部分，也可能出现在后半部分，还有可能出现在中间。
